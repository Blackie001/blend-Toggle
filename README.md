> A hypnotic full-screen web experiment where the word **TOGGLE** stays centered and automatically switches between black and white as you scroll through infinite alternating stripes — using **zero JavaScript**.

![TOGGLE Preview](preview.gif)

## Features

- Fixed centered text that **auto-inverts** on scroll  
- Pure CSS solution using `mix-blend-mode: difference`  
- Smooth, performant, and works on all modern browsers  
- Fully responsive  
- No dependencies · No build tools · Just open and go
